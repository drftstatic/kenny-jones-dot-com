# kennyjones.com

Portfolio website for Kenneth "Kenny" Jones — freelance videographer, documentary shooter, and survivor of the 1996 Woodsboro murders.

Built with Next.js, Tailwind CSS, and Framer Motion. Designed to look like a CRT monitor displaying a modern portfolio site.

## Quick Start

```bash
npm install
npm run dev
```

## Tech Stack

- Next.js 14+ (App Router, TypeScript)
- Tailwind CSS 3.4+
- Framer Motion
- Deployed on Vercel
